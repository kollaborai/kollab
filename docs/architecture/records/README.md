# Architecture Records

This folder holds historical engineering records rather than current
architecture canon.

- `reviews/` contains architecture or UX review writeups.
- `audits/` contains audits, completion checklists, and implementation gap analysis.
- `implementation/` contains public implementation records for completed delivery efforts.
- `test-results/` contains validation snapshots tied to a specific effort.
