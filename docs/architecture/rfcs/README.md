# Architecture RFCs

This folder holds proposed or historically specced architecture changes.

- Put draft or proposed designs here.
- Name RFC files as `RFC-YYYY-MM-DD-slug.md`.
- Use a batch index file when several RFCs were authored together in one session.
- Once a decision is explicitly accepted, capture the durable decision in
  `../decisions/` as an ADR.
- Once an RFC is superseded or intentionally abandoned, move it to
  `../archive/`.
